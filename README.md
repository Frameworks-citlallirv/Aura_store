##  Descripción del Proyecto

**Aura_store** es un sitio web moderno y minimalista enfocado en la venta de accesorios como aretes, collares y pulseras.  
El proyecto busca ofrecer una experiencia visual limpia y fluida, mejorando la forma en que los usuarios exploran y seleccionan productos.

Está desarrollado como una **aplicación web moderna** utilizando un framework de frontend para aprovechar la reactividad, el manejo de estado y la actualización dinámica de la interfaz.


##  Contexto y Justificación

### Problemática
Los emprendimientos de accesorios suelen vender principalmente a través de redes sociales, lo que dificulta:
- Mostrar un catálogo ordenado
- Mantener una imagen de marca profesional
- Ofrecer una experiencia de navegación clara y agradable

Aura_store facilitara la compra de accesorios para los clientes 

### Solución
El uso de un framework de frontend permite:
- Actualizar el catálogo de accesorios sin recargar la página
- Navegar entre productos de forma fluida
- Mantener un diseño minimalista y moderno

Esto mejora la experiencia del usuario y refuerza la identidad visual de la marca.



##  Requerimientos Funcionales

* **RF1:** Visualizar un catálogo interactivo de accesorios.
* **RF2:** Filtrar accesorios por categoría (aretes, collares, pulseras).
* **RF3:** Visualizar el detalle de un accesorio.
* **RF4:** Agregar accesorios a un carrito de compra interactivo.
* **RF5:** Visualizar el carrito con actualización en tiempo real.


##  Requerimientos No Funcionales
* **RNF1:** Adaptabilidad	El sitio se adapta correctamente a celular, tablet y computadora.
* **RNF2:** Rendimiento	Las páginas cargan de forma rápida y fluida.
* **RNF3:** Usabilidad	La navegación es clara, sencilla e intuitiva.
* **RF4:**  El sitio guarda temporalmente la información necesaria para una navegación continua.
